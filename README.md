# Temperature Converter

A simple program to convert between Celsius and Fahrenheit.

## How to run
1. Install Python 3.
2. Run the script:
   ```bash
   python converter.py
   ```

## Example
```
Temperature Converter
Convert from (C)elsius or (F)ahrenheit? C
Enter temperature in Celsius: 25
25°C = 77.00°F
```
